# Real Estate Analytics Pipeline

**Technologies :** Python (Pandas), ETL, Data Cleaning.

## Description
Ce projet simule un pipeline de données complet pour une agence immobilière.
L'objectif est de transformer des données brutes et bruitées en un dataset propre, prêt pour l'analyse décisionnelle (BI).

## Fonctionnalités
- **Génération de données synthétiques** réalistes.
- **Nettoyage (Cleaning)** : Gestion des outliers et valeurs nulles.
- **Feature Engineering** : Calcul du prix au m² et segmentation.
- **Export** : Génération automatique d'un fichier CSV prêt pour Power BI.

## Comment lancer le script
`python etl_pipeline.py`
